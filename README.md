# Palm QGIS Lite (PyQGIS)

Desktop viewer berbasis QGIS engine asli (render cepat, RAM hemat).

## Jalankan
1. Pastikan QGIS 3.40.13 terpasang di `C:\Program Files\QGIS 3.40.13`
2. Jalankan:
```
qgis_app\run_qgis_app.bat
```

## Fitur
- Open GeoTIFF
- Open KML/GeoJSON/SHP/GPKG
- Layer panel
- Clear layers
